# Simulation of Polar Satellite Communication Network using NS2

## Overview
This project simulates a **Polar Satellite Communication Network** using **NS2 (Network Simulator 2)**.
It models communication between satellites and ground stations to study performance metrics such as:
- Throughput
- Packet Delivery Ratio (PDR)
- End-to-End Delay

---

##  Files Included
| File | Description |
|------|--------------|
| `tcl-sat.tcl` | Main NS2 simulation script |
| `sat_trace.tr` | Output trace file generated by NS2 |
| `perf.awk` | AWK script to calculate performance metrics |
| `throughput.plt`, `pdr.plt`, `delay.plt` | Gnuplot scripts for visualization |
| `throughput.dat`, `pdr.dat`, `delay.dat` | Data files for plotting |
| `project.sh` | Optional automation shell script |

---

##  Simulation Command
```bash
ns tcl-sat.tcl 15
awk -f perf.awk sat_trace.tr > result.txt
cat result.txt



```
Here, **15** is the number of satellite nodes used in the simulation.

---

##  Result Visualization
To view results using **Gnuplot**, run:
```bash
gnuplot throughput.plt
gnuplot pdr.plt
gnuplot delay.plt
```
